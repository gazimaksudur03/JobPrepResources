# Coding Test Preparation Guide

> A structured, three-stage roadmap to prepare for C++ coding interviews — from fundamentals to advanced competitive patterns.

---

## How to Use This Guide

This guide is organized into **three progressive stages**. Each stage builds on the previous one, so follow them in order. Every topic page includes:

- **Concept overview** — simplified explanation of the topic
- **Key details** — in-depth breakdown with examples
- **Practice problems** — each with a problem statement, pseudo-code walkthrough, and a complete C++ solution
- **Complexity analysis** — time and space for every solution
- **Practice resources** — curated links for further study

### Recommended Strategy

1. **Start with Basic Stage** — master each topic before moving on. Spend 1–2 days per topic.
2. **Solve every problem on the page first** before looking at the solution. Write pseudo-code, then code.
3. **Re-implement from memory** 24 hours later to solidify understanding.
4. **Move to Standard Stage** once you can solve Basic-level problems in under 20 minutes.
5. **Advanced Stage** is for competitive programming and senior-level interviews.

### Time Estimates

| Stage | Topics | Suggested Pace | Total Time |
|-------|--------|----------------|------------|
| Basic | 12 topics | 1–2 days each | 2–3 weeks |
| Standard | 13 topics | 2–3 days each | 4–5 weeks |
| Advanced | 12 topics | 3–5 days each | 5–6 weeks |

### Problem-Solving Playbook

1. **Clarify** — inputs, outputs, constraints, edge cases, whether order matters
2. **Pattern match** — identify which data structure or algorithm pattern applies
3. **Dry-run** — walk through a small example by hand
4. **Pseudo-code** — outline the approach before writing real code
5. **Implement** — write clean C++ with clear variable names
6. **Analyze** — state Big-O time and space complexity
7. **Test** — check empty input, single element, duplicates, large input

## Quick Reference: Preparation Order

For those who prefer a linear path through all topics:

**Phase 1 — Language and Basics**
C++ Fundamentals → Functions → Arrays → Strings → Pointers → Recursion → Basic OOP → STL Basics

**Phase 2 — Core Problem-Solving**
Complexity → Data Structures → Searching/Sorting → Math → Hashing → Two Pointers → Sliding Window → Binary Search

**Phase 3 — Intermediate Structures and Algorithms**
Linked Lists/Trees/BST → Heap/Trie/Graph → DFS/BFS → Union-Find → Greedy → DP Basics

**Phase 4 — Advanced Topics**
Advanced C++ → Smart Pointers/RAII → Segment/Fenwick Trees → Advanced Graphs → Advanced DP → String Algorithms

**Phase 5 — Mastery**
Number Theory → Combinatorics → Concurrency → Low-Level Design → Hard Patterns

## Daily Practice Routine

| Time | Activity |
|------|----------|
| 15 min | Review yesterday's topic — re-solve one problem from memory |
| 45 min | Study today's new topic — read concepts, understand patterns |
| 60 min | Solve 2–3 problems on the topic — pseudo-code first, then implement |
| 15 min | Review solutions — analyze complexity, note what you learned |
